# House Price Prediction using Linear Regression

## Project Overview

This project predicts house prices using the Linear Regression machine learning algorithm. It uses different house features such as area, bedrooms, bathrooms, parking, and furnishing status to estimate the house price.

## Objective

To build a machine learning model that predicts house prices based on different house features.

## Tech Stack

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

* Housing.csv
* Total Records: 545
* Total Features: 13

## Workflow

1. Import Libraries
2. Load Dataset
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Feature Selection
6. Train-Test Split
7. Model Training using Linear Regression
8. Prediction
9. Model Evaluation

## Results

* R² Score: **0.653**
* Mean Absolute Error (MAE): **970043.40**

## Conclusion

The Linear Regression model successfully predicts house prices with reasonable accuracy. This project demonstrates the complete end-to-end machine learning workflow.
